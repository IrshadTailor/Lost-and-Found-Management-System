# Lost & Found Management System

A web-based Lost & Found Management System that helps users report, search, and manage lost and found items efficiently.

## Features
- Report lost items
- Report found items
- Search for items
- User authentication (Login/Register)
- Organized item details view

## Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS, JavaScript

## Project Structure
- app.py → Main application file
- templates/ → HTML pages
- static/ → CSS, JS, images
- instance/ → Database files
